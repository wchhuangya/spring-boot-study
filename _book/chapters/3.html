三、使用 Spring Boot

三、使用 Spring Boot

本节将详细介绍如何使用Spring Boot。它涵盖了构建系统，自动配置以及如何运行应用程序等主题。我们还介绍了一些 Spring Boot 的最佳实践。尽管 Spring Boot 没有特别的特殊之处（它只是您可以使用的另一个库），但有还是有一些建议的，如果能遵循这些建议，那你的开发过程会更容易一些。

如果您刚开始使用 Spring Boot，那么在深入本节之前，您应该阅读入门指南。
