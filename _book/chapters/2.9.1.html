9.1 servlet 容器

9.1 Servlet 容器

Spring Boot 支持下列的内嵌 servlet 容器：

Name
Servlet Version
Tomcat 8.5 3.1
Jetty 9.4 3.1
Undertow 1.4 3.1

当然，也可以把 Spring Boot 应用部署到任意支持 servlet 3.1+ 的容器中。
